# Security Policy

## Reporting A Vulnerability

If you believe you have found a security issue, please report it privately by email:

- kiefer.waight@uta.edu

Please include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Any suggested remediation

Do not open public issues for unpatched security findings.

## Scope Notes

This repository focuses on documentation, synthetic data, and demonstration code. Even so, we treat security reports seriously.
